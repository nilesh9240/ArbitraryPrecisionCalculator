Description :
	1)I have used two singly linked null terminated lists to store characteristic part and decimal part
	separately.
	2)each node of these lists is been typedefed as digit,  containing next pointer and a character.
	3)Pointer to these two list and a integer is been typedefed to a user defined type named "number"
	4)Integer in the structure represents sign of the number, 1 for -ve and 0 for +ve.
	5)Each single digit of the number is stored as a character.
	
operations implemented:
	+ , - , <<, power, mod, unary minus.
